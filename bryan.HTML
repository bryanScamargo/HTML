surgimento da internet

Tudo começou com o britânico Berners-Lee que viu a oportunidade de unir hipertexto com TCP\IP dessa forma criando o Word Wide Web (WWW) o primero navegador.Para criar o WWW o Tim Berners-Lee precisou usar o HTML que significa HiperText Markup Lenguage, traduzindo: Linguagem de Marcaçâo de Hipetexto, um componente basico da web.

O html estabelece uma estrutura básica de um website e também serve para dar um significado e organizar informações de uma página web, sem o HTML os navegadores não conseguiria exibir textos, imagens e outros conteúdos
