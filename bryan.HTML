Surgimento da internet

Tudo começou com o britânico Tim Berners-Lee que viu a oportunidade de unir hipertexto com TCP\IP dessa forma criando o Word Wide Web (WWW) o primero navegador. Para criar o WWW o Tim Berners-Lee precisou usar o HTML que significa HiperText Markup Language, traduzindo: Linguagem de Marcação de Hipetexto, um componente básico da web.

O html estabelece uma estrutura básica de um website e também serve para dar um significado e organizar informações de uma página web, sem o HTML os navegadores não consegueria exibir textos, imagens e outros conteúdos, o HTML se baseia em Clients e Server.

Clients

Os Clients ou clientes é a forma de acesso de navegadores, ou seja, celulares, computadores, nootbooks, tablets e etc. Então o client é nada mais nada menos do que a forma de acesso.

Servers

Os servers ou servidores são computadores que tem a função de servir informações ou armazenar para os clientes, ou seja, eles guardam e distribuem informações para um determinado número de clientes. (lembrando que quanto mais servidores mais o limite de clientes aumenta)

Tipos de servidores

  • Arquivos
  • segurança
  • Streaming
  • E-mail
  • web

O que é HTML

(voltar)

O HTML não é uma linguagem de progamação e sim um arquivo marcação, os textos são delimitados por nomes que o navegador consegue interpretar cada determinador tem sua função e compartimento específico. Então progamar em HTML não significa progamar usando uma linguagem de progamação e sim progamar mais na parte textual, por esse motivo, o HTML significa linguagem de marcação de hipertexto

O HTML funcina a base de Tags que são basicamente códigos do HTML, ou seja, as tags servem para progamar são como "atalhos". O HTML pode ser visualizado nos navegadores: Google Chrome, Safari e Mobzila Firefox.

Tags

(voltar)

Conforme explicado anteriormente, o HTML consiste em uma série de tags que controlam e alteram a estrutura da página como: Tamanho; Fonte; cores; imagens; etc.

Algumas das tags são:

  • (h1)
  • (h2)
  • (p)
  • (i)
  • (a)
  • (strong)
  • (u)
  • (mark)
  • (sup)
  • (ol)
  • (ul)
  • (li)
  • (title)
  • (hr)
  • (small)

O que cada tag faz

  • H1: essa tag é usada para títulos, ela deixa a letra maior.
  • H2: Essa tag é normalmente usada em subtítulos, ela deixa a letra maior mas não igual a tag anterior.
  • P: A tag "p" é usada para escrever paragrafos
  • I: A tag "i" é usada para deixar as letras em ítalico, ou seja, deixar as letras deitadas
  • A: A tag "a" é usada como âncora, ela serve para criar links.
  • STRONG: A tag "strong" é usada para deixar as palavras em negrito.
  • U: A tag "u" é usada para sublinhar as frases, ou seja, traçar uma linha embaixo de uma palavra.
  • MARK: A tag "mark" é usada como marca texto, ela deixa a palavra marcada em amarelo.
  • SUP: Essa tag é usada para elevar um número ou uma letra e a deichano pequena igual na matemática.
  • OL: A tag "ol" é usada para fazer listas ordenadas marcadas por números.
  • UL: A tag "ul" é usada para fazer listas não ordenadas marcadas por pontos.
  • LI: Junto com a tag "ol" ou "ul" essa tag de apoio para as outras duas tags.
  • HR: essa tag traça uma linha para separar linhas de palavras.
  • SMALL: A tag "small" faz com que deixe as palavras pequenas .
